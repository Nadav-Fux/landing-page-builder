# תוכנית פיתוח מפורטת - מערכת התבניות (גרסה 2)

## סקירה כללית
תוכנית פיתוח מעשית ומפורטת למערכת התבניות, מחולקת למשימות קטנות של 30-60 דקות כל אחת.

## שלב 1: תיקון שגיאות נוכחיות (60 דקות)

### משימה 1.1: תיקון שגיאות Client Components (30 דקות)
**מטרה:** תיקון שגיאות Next.js הקשורות ל-client components

**צעדים מדויקים:**
1. הוסף `"use client"` לתחילת הקובץ `src/components/ui/ToastContainer.tsx`
2. הוסף `"use client"` לתחילת הקובץ `src/hooks/useToast.ts`
3. בדוק אם יש קבצים נוספים שזקוקים ל-directive זה
4. הפעל `npm run dev` ובדוק שאין שגיאות

**תוצאה צפויה:** השרת רץ ללא שגיאות

### משימה 1.2: וידוא תקינות המערכת (30 דקות)
**מטרה:** וידוא שכל המערכת עובדת תקין

**צעדים מדויקים:**
1. פתח את http://localhost:3000
2. נווט לעמוד העורך
3. בדוק שכל הבלוקים נטענים כראוי
4. בדוק שניתן לגרור ולשחרר בלוקים
5. בדוק שהשמירה עובדת

**תוצאה צפויה:** מערכת יציבה ופועלת

## שלב 2: שיפור תבנית Modern Business (90 דקות)

### משימה 2.1: הוספת סקשן About Us (30 דקות)
**מטרה:** הוספת סקשן "אודותינו" מקצועי

**צעדים מדויקים:**
1. פתח את `src/app/editor/page.tsx`
2. מצא את מערך הבלוקים ב-`blockManager`
3. הוסף בלוק חדש עם id: `about-us`
4. תוכן הבלוק:
   - כותרת "אודותינו"
   - תמונה placeholder
   - 2-3 פסקאות טקסט
   - רשימת ערכים/יתרונות

**קוד לדוגמה:**
```javascript
{
  id: "about-us",
  label: "<div><i class='fa fa-users'></i><div>אודותינו</div></div>",
  content: `<section class="py-16 px-4 bg-white">...</section>`
}
```

### משימה 2.2: הוספת סקשן Services (30 דקות)
**מטרה:** הוספת סקשן שירותים עם grid

**צעדים מדויקים:**
1. הוסף בלוק חדש עם id: `services-grid`
2. תוכן הבלוק:
   - כותרת "השירותים שלנו"
   - Grid של 3 עמודות
   - כל שירות עם אייקון, כותרת ותיאור
   - עיצוב responsive

### משימה 2.3: הוספת סקשן Testimonials (30 דקות)
**מטרה:** הוספת סקשן המלצות לקוחות

**צעדים מדויקים:**
1. הוסף בלוק חדש עם id: `testimonials`
2. תוכן הבלוק:
   - כותרת "מה הלקוחות אומרים"
   - 3 כרטיסי המלצות
   - כל כרטיס עם תמונה, שם, תפקיד וציטוט
   - עיצוב עם shadows וגבולות מעוגלים
