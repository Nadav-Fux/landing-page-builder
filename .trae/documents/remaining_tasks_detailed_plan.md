# תוכנית מפורטת למשימות הנותרות - Landing Page Builder

## סקירה כללית
תוכנית פיתוח מעשית ומפורטת למשימות שעוד לא הושלמו, מחולקת לשלבים קטנים של 30-90 דקות כל אחת.

---

## שלב 1: רקע אינטראקטיבי לעמוד הבית (עדיפות גבוהה) - 4 שעות

### משימה 1.1: יצירת קומפוננטת הרקע הבסיסית (60 דקות)
**מטרה:** יצירת קומפוננטה נפרדת לרקע אינטראקטיבי

**צעדים מדויקים:**
1. צור קובץ `src/components/ui/InteractiveBackground.tsx`
2. צור interface לתצורת הרקע:
```typescript
interface BackgroundConfig {
  id: string;
  name: string;
  component: React.ComponentType;
  duration: number;
}
```
3. צור hook `useBackgroundManager` לניהול החלפת הרקעים
4. הוסף state לניהול הרקע הנוכחי והרקע הקודם
5. הוסף אלגוריתם למניעת חזרה על אותו רקע

**תוצאה צפויה:** קומפוננטה בסיסית עם מבנה לניהול רקעים

### משימה 1.2: יצירת 5 רקעים ראשונים (90 דקות)
**מטרה:** יצירת חצי מהרקעים הנדרשים

**רקעים ליצירה:**
1. **ParticleFlow** - חלקיקים צבעוניים נעים
2. **DigitalWaves** - גלים דיגיטליים זורמים
3. **NetworkGrid** - רשת משתנה של קווים
4. **AuroraLights** - אורות צפוניים דיגיטליים
5. **PixelBurst** - פיקסלים מתפוצצים ומתאחדים

**צעדים לכל רקע:**
- צור קומפוננטה נפרדת ב-`src/components/backgrounds/`
- השתמש ב-CSS animations או Framer Motion
- וודא ביצועים טובים עם `transform` ו-`opacity`
- הוסף responsive design

### משימה 1.3: יצירת 5 רקעים נוספים (90 דקות)
**מטרה:** השלמת כל הרקעים

**רקעים ליצירה:**
6. **FlowingGradients** - גרדיאנטים נעים ומשתנים
7. **GeometricShapes** - צורות גיאומטריות מסתובבות
8. **MatrixCode** - אפקט מטריקס עם קוד זורם
9. **StarField** - כוכבים נוצצים ברקע חלל
10. **ElectricEnergy** - אנרגיה חשמלית זורמת

### משימה 1.4: הוספת אינטראקטיביות (60 דקות)
**מטרה:** הוספת אפקטים בקליק על הרקע

**צעדים מדויקים:**
1. הוסף event listener לקליקים על הרקע
2. צור 5 אפקטי קליק שונים:
   - פיצוץ צבעוני במקום הקליק
   - גלי אנרגיה מהמרכז החוצה
   - שינוי זמני של הרקע
   - אפקט "ריפל" על פני המסך
   - הופעת אלמנטים מיוחדים זמניים
3. הוסף אנימציות CSS לכל אפקט
4. וודא שהאפקטים לא משפיעים על ביצועים

### משימה 1.5: אינטגרציה בעמוד הבית (30 דקות)
**מטרה:** הוספת הקומפוננטה לעמוד הבית עם אפשרות הפעלה/כיבוי

**צעדים מדויקים:**
1. הוסף toggle ב-localStorage לשמירת העדפת המשתמש
2. הוסף כפתור קטן בפינה להפעלה/כיבוי
3. הוסף את הקומפוננטה כ-background לעמוד הבית
4. וודא שהתוכן נשאר קריא מעל הרקע
