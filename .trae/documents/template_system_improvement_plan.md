# תוכנית שיפור מערכת התבניות - משימות קטנות ומנוהלות

## סקירה כללית
תוכנית זו מחלקת את העבודה למשימות קטנות ועצמאיות, כל אחת ניתנת לביצוע תוך 5-10 דקות.

## שלב 1: יצירת תבניות נוספות (עדיפות גבוהה)

### 1.1 תבנית חנות מקוונת
**זמן משוער:** 8 דקות
**תיאור:** יצירת תבנית מלאה לחנות מקוונת עם קטלוג מוצרים
**משימה:**
- הוספת תבנית 'ecommerce-store' ל-`src/data/templates.ts`
- כולל HTML מלא עם: header, hero, קטלוג מוצרים, עגלת קניות, footer
- CSS מתקדם עם אנימציות hover ו-responsive design
- תוכן בעברית עם תמיכה RTL
- צבעים: כחול-ירוק (#2563eb, #10b981)

### 1.2 תבנית מסעדה
**זמן משוער:** 8 דקות
**תיאור:** יצירת תבנית לאתר מסעדה עם תפריט ויצירת קשר
**משימה:**
- הוספת תבנית 'restaurant-menu' ל-`src/data/templates.ts`
- כולל HTML מלא עם: navigation, hero, תפריט, גלריה, הזמנות, יצירת קשר
- CSS עם אפקטים ויזואליים ואנימציות
- תוכן בעברית עם תמיכה RTL
- צבעים: חום-זהב (#8b4513, #ffd700)

### 1.3 תבנית טכנולוגיה
**זמן משוער:** 8 דקות
**תיאור:** יצירת תבנית לחברת טכנולוגיה/סטארטאפ
**משימה:**
- הוספת תבנית 'tech-startup' ל-`src/data/templates.ts`
- כולל HTML מלא עם: header מודרני, hero טכנולוגי, תכונות, צוות, השקעות
- CSS מתקדם עם gradients ואנימציות
- תוכן בעברית עם תמיכה RTL
- צבעים: סגול-כחול (#6366f1, #3b82f6)

### 1.4 תבנית בריאות ורפואה
**זמן משוער:** 8 דקות
**תיאור:** יצירת תבנית לקליניקה או שירותי בריאות
**משימה:**
- הוספת תבנית 'health-wellness' ל-`src/data/templates.ts`

- תוכן בעברית עם תמיכה RTL
- צבעים: ירוק-כחול רפואי (#059669, #0284c7)

## שלב 3: פיצ'רים מתקדמים (עדיפות בינונית)
